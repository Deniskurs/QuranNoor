## Design Philosophy

<br>

> *"Simplicity is the ultimate sophistication."*

Qur'an Noor embraces a design language inspired by tranquility. The color palette draws from the emerald domes of Madinah, the golden calligraphy of the Quran, and the deep midnight sky under which the first revelations were received.

<br>

| | |
|---|---|
| **Emerald** `#0D7377` | The primary accent â€” evoking the sacred |
| **Gold** `#C7A566` | Warmth and reverence in every highlight |
| **Midnight** `#1A2332` | Depth and focus for dark themes |
| **Cream** `#F8F4EA` | Soft, paper-like warmth for light reading |

<br>

Four reading themes â€” **Light**, **Dark**, **Night**, and **Sepia** â€” each carefully tuned for extended reading comfort. Dynamic Type support ensures the app adapts to every reader.

<br>

## Architecture

Built entirely in **Swift** and **SwiftUI**, Qur'an Noor follows a clean **MVVM** architecture with modern iOS patterns:

```
MVVM + @Observable + SwiftData + Combine
```

- **SwiftData** for persistent storage with versioned schema migrations
- **@Observable** macro for reactive, performant state management
- **Async/Await** and structured concurrency throughout
- **Centralized audio session management** for seamless playback
- **Offline-first** design â€” core features work without internet
